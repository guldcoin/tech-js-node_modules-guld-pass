# guld-pass

Encrypted password management comaptible with Standard Unix Password Store.

### Install

```
npm i guld-pass
```

### Usage

##### Node

```
const { init, show, insert } = require('guld-pass')
```

##### CLI

See [guld-uesr-cli](https://github.com/isysd/tech-js-node_modules-guld-pass-cli).
