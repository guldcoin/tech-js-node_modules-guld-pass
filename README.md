# guld-user

Guld user management tools. Get, list, and check users of the guld group.

### Install

```
npm i guld-user
```

### Usage

##### Node

```
const { getName, getFullName } = require('guld-user')
```

##### CLI

See [guld-uesr-cli](https://github.com/isysd/tech-js-node_modules-guld-uesr-cli).
